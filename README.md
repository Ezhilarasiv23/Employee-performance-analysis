# Performance Appraisal System Study

## Project Overview
This project analyzes the effectiveness of the performance appraisal system and employee satisfaction at Super Med Pharmacy Limited.

## Objective
- To study employee satisfaction
- To analyze appraisal effectiveness
- To evaluate employee motivation
- To examine organizational performance

## Research Methodology
- Sample Size: 100 Employees
- Sampling Technique: Simple Random Sampling
- Tools Used:
  - Percentage Analysis
  - ANOVA
  - Chi-Square Test
  - F-Test

## Company Profile
The study was conducted at Super Med Pharmacy Limited, Chennai.

## Key Findings
- Employees positively viewed the appraisal system
- Appraisal improves motivation and satisfaction
- Transparent feedback increases employee trust
- Appraisal contributes to organizational performance
